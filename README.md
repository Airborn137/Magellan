# Magellan

**Magellan** — это исследовательское iOS-приложение с уникальной механикой: интерактивная карта раскрывает окружающий мир только в радиусе 50 метров от пользователя. Все остальное покрыто "туманом войны".

### Основные фичи:
- Отображение только ближайшей зоны на карте
- Пошаговое открытие новых территорий
- История передвижений и достижений
- Геймификация: уровни, ачивки, открытия

### Технологии:
- Swift 5
- SwiftUI
- CoreLocation
- MapKit

---

**Цель проекта:** стимулировать пользователей исследовать своё окружение, превратив повседневную прогулку в приключение.
